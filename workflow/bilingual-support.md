Toggle between English and Traditional Chinese. The household uses both languages depending on context.

## User Stories

- As a user, I want to switch the app language so I can use the interface in my preferred language.

## Success

- Language toggle in settings or nav
- All UI strings translated: labels, buttons, categories, navigation, error messages
- Language preference persists per user session
- Category names are bilingual (stored in DB)

### Out of Scope

Auto-detection from browser locale. Right-to-left language support.
